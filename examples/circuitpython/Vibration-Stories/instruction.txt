Background
You are sending values for a vibration motor to send a vibration melody.
A variable‑length vibration pattern is just a list of vibration “notes.”
Each note is a triple: [intensity, duration, pause].
intensity: how strong the vibration is (e.g. 0–100).
duration: how long that vibration lasts (in seconds).
pause: how long to stay quiet before the next vibration.
Because the list can be any length, you can have one note (a single buzz), a few notes (like “buzz‑buzz‑buzz”), or many notes (a long rhythm), for example:
[[100, 0.5, 0.5]] → one strong half‑second buzz, then stop
[[100, 0.5, 0.5], [80, 0.2, 0.3]] → a strong buzz, short pause, then a softer, quicker buzz
A longer list creates a more complex vibration “melody.”
